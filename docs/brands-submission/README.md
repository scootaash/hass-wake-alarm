# home-assistant/brands submission for `wake_alarm`

Staged icons for the Wake Alarm entry in
https://github.com/home-assistant/brands — required before submitting the
integration to the HACS **default** store (so the icon resolves in the HA/HACS
UI).

## Files
- `custom_integrations/wake_alarm/icon.png` — 256×256, square, trimmed, RGBA
- `custom_integrations/wake_alarm/icon@2x.png` — 512×512, square, trimmed, RGBA

Generated by scaling `custom_components/wake_alarm/brand/icon@2x.png` (315×323)
to an exact square — a ~2.5% aspect adjustment to satisfy the brands repo's
"square + trimmed + exact size" rules. Swap in purpose-built square artwork if
you'd prefer to avoid the slight stretch.

## How to submit
1. Fork https://github.com/home-assistant/brands from your **personal** account.
2. Copy the `custom_integrations/wake_alarm/` folder (here) into the fork root.
3. Commit on a new branch off `master` and open a PR. Brands CI validates the
   PNG size / square / trim automatically.

(Optional: add `logo.png` / `logo@2x.png` for a wordmark — not required.)
